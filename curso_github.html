Curso de Git y Github con Platzi

Comando en GIT

pwd

Te ubica en la carpeta actual

cd

Te cambia de directorio

pwd

Te ubica en la carpeta actual

mkdir

Te crea un nuevo directorio o carpeta en la carpeta acutla

touch

Te crea nuevos acrhivos en la carpeta actual

ls

Te da una lista de los archivos que se encuentran en la carpeta actual

cat "archivo"

Te mostrara el contenido del archivo

history

Te muestra los comandos que usaste

rm "archivo/directorio"

Te borra el archivo o directorio mencionado

git init

Te crea el repositorio de git en la carpeta actual

git status

Te da el estatus de tus archivos

code

Te invoca a Visual Studio Code

git add "archivo"

Te añade el archivo a taging para trackearlos

git commit -m

Te añade los cambios al repositorio creando la versio del archivo trackeado

git log "archivo"

Te muestra todos los commits realizados

git show "archivo"

Te muestra detalladamente los cambios que hiciste en tu archivo de texto

git dif "codecommit3" "codecommit8"

Te identifica las diferentes cambios entre un commit a y un commit b usando los codigos obtenidos en git logs. (colocar la version mas vieja de primero que la nueva)

git rest "codecommitx"

Te regenera esa version mencionada como la ultima, es decir como el "HEAD". ATENCION: Este comando puede borrar TODOS LOS CAMBIOS que hayas realizado, ten precaucion.

git log --stat

Te identifica los cambios especificos hechos en cada uno de los commits.
